# Smart Water Quality Monitoring

## Overview

This repository contains the implementation of an IoT-based water quality monitoring system built using an ESP32 microcontroller. The project was developed to demonstrate real-time monitoring, cloud-based visualization, and basic contamination detection using low-cost embedded hardware.

## Purpose of This Repository

This repository exists to:

* Document the design and implementation of a smart water monitoring prototype
* Provide working firmware for ESP32 sensor integration
* Demonstrate cloud data logging using ThingSpeak
* Showcase real-time anomaly detection logic
* Serve as a reference for academic, research, or industrial IoT applications

## System Features

* pH monitoring using analog sensor
* Water temperature measurement (DS18B20)
* Water level monitoring using ultrasonic sensor
* Wi-Fi-based cloud data upload
* Threshold-based contamination alerts
* Real-time graphical visualization on ThingSpeak

## Technology Stack

* ESP32 (Wi-Fi-enabled microcontroller)
* Arduino framework (C/C++)
* ThingSpeak Cloud Platform
* HTTP-based data transmission

## Repository Structure

* `main.ino` / firmware file – Complete ESP32 implementation
* Documentation – Project report and system explanation
